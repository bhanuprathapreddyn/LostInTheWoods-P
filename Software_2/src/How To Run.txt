run menu.py

A menu page should be displayed

Inside the menu page there will be
    1. start
    2. option

    for start page:
        user can go back to menu by press "esc" key
        user can select the game mode by clicking the button:
            once user select the game mode and clicked the button, game will start

    for option page:
        user can also go back to menu by press "esc" key
        user can change the volume setting mode by clicking the button


How to control:
  Jack(user 1):    is able to press keyboard "w", "s", "a", "d" to move up, down, left, right
  Oggy(user 2):  is able to press keyboard up, down, left, right to move up, down, left, right
  Sandy(user 3):      is able to press keyboard "1", "2", "3", "4" to move up, down, left, right
  Squid(user 4):      is able to press keyboard "6", "7", "8", "9" to move up, down, left, right


How to win:
    when Jack meet Oggy, the game will be over
